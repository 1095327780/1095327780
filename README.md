## 关于我

你好，我是阿浩。现在主要在做 AI 原生的个人知识管理、笔记工作流和效率工具。

我长期使用 Obsidian、Zettelkasten、PARA 和项目制工作流，也会把真实使用中的问题做成插件、自动化脚本、iOS 应用和内容教程。相比只做一个单点工具，我更关心一件事：如何让记录、整理、连接、行动和复盘真正形成一条可持续的流水线。

目前重点方向：

- 在 Obsidian 里构建更顺手的 AI 笔记与技能工作流。
- 把日记、阅读、任务、项目和复盘串成可执行系统。
- 探索 AI Agent、内置工具调用、多端知识库和个人自动化。
- 通过 B 站和公众号分享知识管理、效率工具、AI 工作流的实践过程。
